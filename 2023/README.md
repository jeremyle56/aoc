# Example use of Template

## Folder Creation

```bash
$ ./newday.sh day24 # Add "&& code day24" if you also want to open it in Visual Studio Code
$ ./newday_py.sh day24 # For Python template
```

## Program Compilation

```bash
$ cd day24
$ make # Only for C++
```

## Running Code on Input

Edit `input.txt` with your input.

```bash
$ ./day24
```
